# PIONEERS!! - Ring Catcher Game

하늘에서 떨어지는 고리를 막대로 받는 게임입니다.

## 설치 및 실행

```bash
npm install
npm run dev
```

`http://localhost:3000` 에서 게임을 즐기세요.

## GitHub Pages 배포

```bash
git push origin main
```

GitHub Actions가 자동으로 `https://yourusername.github.io/pioneers-ring-catcher` 에 배포합니다.

## 게임 조작

- 마우스 드래그로 막대 이동
- 떨어지는 고리를 받아 점수 획득
- 5번 놓치면 게임 오버
- 2000점 또는 100개 달성 시 승리
